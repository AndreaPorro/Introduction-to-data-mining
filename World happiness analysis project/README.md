# 🌍 Global Wellbeing: World Happiness Report Analysis

This repository presents a detailed statistical analysis of the **World Happiness Report**.

The project investigates how economic, social, and health factors contribute to the happiness score of different countries, comparing data across multiple years.

## 🎯 Project Objectives
The analysis focuses on identifying the key drivers of global happiness:
* **Comparative Analysis:** Comparing the 2021 report with the updated 2024 data to track changes in global wellbeing.
* **Feature Exploration:** Analyzing the impact of GDP per capita, Social Support, Healthy Life Expectancy, and Freedom to make life choices.
* **Statistical Insights:** Using correlation and regression techniques to understand which variables have the strongest influence on the "Ladder Score" (Happiness score).

## 🛠️ Tech Stack & Skills
* **Language:** `R`
* **Reporting:** `R Markdown`
* **Key Libraries:** `tidyverse`, `ggplot2`, `corrplot`, `plotly`.
* **Methods:** Exploratory Data Analysis (EDA), Correlation Analysis, Time-series Comparison, Data Visualization.

## 📈 Key Insights & Results
* **Economic Impact:** GDP per capita remains one of the strongest predictors of happiness, showing a high positive correlation with the overall score.
* **Social Factors:** Social support and healthy life expectancy were identified as the second most influential factors for top-ranking countries.
* **Global Trends:** The analysis highlights a significant stability in the top-ranking countries (e.g., Nordic countries) while showing fluctuations in developing regions based on social and economic changes between 2021 and 2024.

## 📂 Repository Structure
* `Wold happiness analysis.R`: Clean R script for the analytical pipeline.
* `World-happiness-analysis.Rmd`: The complete R Markdown script with data processing and visualizations.
* `World-happiness-analysis.pdf`: The final technical report.
* `world-happiness-report-2021.csv`: Historical data for comparison.
* `World-happiness-report-updated_2024.csv`: The most recent dataset analyzed.
